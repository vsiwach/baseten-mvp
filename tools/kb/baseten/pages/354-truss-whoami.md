# truss.whoami
Source: https://docs.baseten.co/reference/sdk/truss/whoami



Returns account information for the authenticated user.

**Parameters:**

| Name     | Type             | Description                                                                             |
| -------- | ---------------- | --------------------------------------------------------------------------------------- |
| `remote` | *Optional\[str]* | Name of the remote in `.trussrc` to query. Inferred when only one remote is configured. |

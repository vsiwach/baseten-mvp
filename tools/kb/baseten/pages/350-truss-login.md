# truss.login
Source: https://docs.baseten.co/reference/sdk/truss/login



Authenticates with Baseten using an API key. Persists credentials to `~/.trussrc`, so it only needs to be called once.

**Parameters:**

| Name      | Type  | Description     |
| --------- | ----- | --------------- |
| `api_key` | *str* | Baseten API Key |
